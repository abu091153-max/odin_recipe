# odin_recipe

A simple recipe website built as part of The Odin Project Foundations curriculum.

## Description

This project is a basic HTML website that consists of a main index page and several recipe pages. The purpose of this project is to practice structuring content with HTML, using links and images, and organizing a small multi-page website.

## Features

- Homepage with links to individual recipes
- Multiple recipe pages
- Images for each recipe
- Proper HTML structure and semantics

## Technologies Used

- HTML5

## What I Learned

- How to structure a basic website using only HTML
- How to use relative links between pages
- How to organize files and folders in a project
- How to use Git and GitHub for version control
- How to publish a site using GitHub Pages
